# Recipe App

A simple web application to search for recipes and view meal details.

Users can search for their favorite recipes, view results in cards, and open a modal to see recipe information.

## Features

- Search recipes by name
- Fetch data from external API
- Display recipes in responsive cards
- View recipe details in a modal
- Handle loading and empty states

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Google Fonts
- TheMealDB API

## API Used

TheMealDB API
